Site Name

This site's name will be Weather Me This, and its domain will be weathermethis.com which is currently available. This is an ideal domain because it is short, uses a popular ending (.com), and would only cost $12/year.

This website will be a great place to check the weather. This site will be used as a tool by everyone from individuals who need a quick weather update, to organizations planning events.
